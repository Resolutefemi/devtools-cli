## ⚡ Quick Start

### Installation
Install globally via pip:
```bash
pip install renance-dt
```

### Automatic Setup
Configure your system PATH automatically so you can use the `dt` command from anywhere:
```bash
dt setup
```
*Note: Please restart your terminal after running setup.*

## 🛠️ Key Categories (200+ Commands)

Renance DevTools is organized into 12 powerful categories. Run `dt help` to see the full dashboard.

### 🚄 1. High-Speed Files (`dt fcp`, `dt send`, `dt big`)
*   **`dt fcp`**: Multi-threaded copy engine for moving massive folders up to 5x faster.
*   **`dt search`**: Recursive "super-search" inside file contents.
*   **`dt big`**: Instantly find the top 10 largest files clogging your drive.

### 🎬 2. Media Processing (`dt join`, `dt shrink`, `dt music`)
*   **`dt shrink`**: Compress videos for WhatsApp or Web.
*   **`dt music`**: Extract high-quality MP3s from any video file.
*   **`dt gif`**: Convert video clips into optimized GIFs.

### 🌐 3. Network & Deployment (`dt ship`, `dt speed`, `dt whois`)
*   **`dt ship`**: Interactive deployment engine for **Vercel, Netlify, and Render**.
*   **`dt speed`**: Accurate internet speed test (Down/Up/Ping).
*   **`dt status`**: Check if a website is globally accessible.

### 🕶️ 4. Hacker Mode (`dt matrix`, `dt vault`, `dt sniff`)
*   **`dt matrix`**: Iconic falling green code effect.
*   **`dt vault`**: Encrypt/Decrypt sensitive files with a custom password.
*   **`dt port-scan`**: Scan any host for open communication ports.

### 🧮 5. Math, Text & Crypto
*   **Text**: `upper`, `lower`, `slugify`, `morse`, `camelcase`, `reverse`...
*   **Math**: `add`, `sqrt`, `bmi`, `c2f`, `fact`, `sin`, `cos`...
*   **Crypto**: `passgen`, `hash`, `b64enc`, `hexdec`, `uuid`...

### 🤡 6. Fun & Utilities
*   **Fun**: `chuck`, `catfact`, `joke`, `magic8`, `rps`, `bitcoin`...
*   **Utils**: `pomo` (Pomodoro timer), `weather`, `qr`, `todo`, `shorten`...

## 📖 Usage

### Dashboard
To see the interactive dashboard and all available categories:
```bash
dt help
```

### About the Project
To see credits and project information:
```bash
dt about
```

### Command Help
To get specific instructions for any command:
```bash
dt [command] --help
```
